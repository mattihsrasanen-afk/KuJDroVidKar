Suomeksi: Projektin kuvaus

Tämä työkalu visualisoi kuva- ja videomateriaaliasi maantieteellisesti hyödyntämällä Maanmittauslaitoksen (MML) avoimia kartta-aineistoja. Ohjelma skannaa valitsemasi hakemistot ja sijoittaa GPS-metadataa sisältävät tiedostot automaattisesti oikeisiin koordinaatteihin kartalla.

Tärkeimmät ominaisuudet:

    Automaattinen paikannus: Lukee GPS-tiedot suoraan kuvista ja videoista.

    Aikajanan hallinta: Materiaali jaotellaan eri tasoille (layers) valmistumisvuoden mukaan, mikä tekee selaamisesta selkeää.

    Tarkat karttapohjat: Hyödyntää Maanmittauslaitoksen rajapintoja korkealaatuisen taustakartan tarjoamiseksi.

In English: Project Overview

This tool visualizes your photos and videos geographically by utilizing open-source map data from the National Land Survey of Finland (Maanmittauslaitos). The program scans specified directories and automatically plots files with GPS metadata onto the map.

Key Features:

    Automatic Geolocation: Extracts GPS coordinates directly from image and video metadata.

    Chronological Layers: Organizes media into map layers based on the year taken, providing a clean and organized user experience.

    High-Quality Maps: Integrates with official Finnish topographic maps and datasets.
    
Tarkemmat ohjeet löytyvät OHJEET.txt tiedostosta.
